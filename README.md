# Google Maps Scraper

Ce projet permet de récupérer automatiquement des informations (nom, téléphone, adresse...) d’établissements sur Google Maps en fonction de différentes villes, puis de les enregistrer dans un fichier CSV.

- Scraping automatisé
- Multi-villes
- Anti-doublons
- Export CSV

Lancez simplement :  
```bash
node script.js
